2. Dashboard Layout

The image above shows the dashboard of Fugaku Open OnDemand. The meanings of the items in the top menu bar are as follows (some may switch to icons if the browser window is narrow):

ItemDescription
Batch JobsBatch jobs submitted to Fugaku or the Prepost environment
Interactive AppsInteractive applications running on Fugaku or the Prepost environment
Passenger AppsApplications running on the Open OnDemand server
My Interactive SessionsA list of session information for interactive applications
Help → Restart Web ServerRestart Open OnDemand
Help → ProfilesClick "All apps" to display all icons for Batch Jobs and Interactive Apps on the dashboard. Click "Default" to revert.

The numbers in the image correspond to the following items:

  1. External Links
  2. Latest News
  3. Queue Status in Job Scheduler
  4. Operational Schedule
  5. Disk and Budget Usage
  6. Recently Used Interactive Applications (up to 4)
  7. Passenger Apps

Applications can be launched from the menu bar or from the icons in items 6 and 7.
