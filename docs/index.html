1. Introduction

Open OnDemand is a web portal that allows users to access supercomputers via a web browser. Users can perform the following actions via their browser:

  • Upload, download, and edit files
  • Use the terminal
  • Submit and manage interactive applications (such as a remote desktop) and batch jobs
  • View user information (e.g., budget and disk usage)

Open OnDemand supports major web browsers including Google Chrome, Mozilla Firefox, and Microsoft Edge (Note: Internet Explorer 11 is not supported). Among these, Chrome is recommended as it natively supports copy and paste functionality in the remote desktop.
