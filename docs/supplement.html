6. Supplement

  • The temporary directory for Open OnDemand is ${HOME}/ondemand. It is automatically created when using Open OnDemand. This directory stores things like extensions installed in interactive applications such as JupyterLab, and usage history.
  • The temporary directory for batch jobs is ${HOME}/composer. It is automatically created when using batch jobs. It stores batch job history and related data.
  • If a problem occurs, please click "Help" → "Restart Web Server" in the menu bar. If the issue persists, please contact the Fugaku Support Site.
  • If you would like to request the addition of an interactive application or batch job, please also contact the Fugaku Support Site.
